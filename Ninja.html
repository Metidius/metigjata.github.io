Ninja (C# & Unity)

Ninja is a Casual Free-to-Play Game with a variety of Game Modes in which the player must jump from tile to tile to advance through the level whilst avoiding the deadly lava below.

I have been working on The Floor is Lava since mid-2016 when it initially started as part of my 3rd Year Project at Brunel University London.

After taking some time off from the project after graduation, I evaluated the scope and size of the project and started to actively work on the project again, finishing it in the winter of 2017. I have plans to release it on the Android Play Store in 2018.

As I was the only designer throughout this project (apart from the art, which was provided by Kenney) I oversaw all elements of design, but highlights include:

  • Early stages of concept development using Paper Prototyping and Clickteam Fusion 2.5 to refine the early mechanics and features to make sure the core mechanics were fun to play and engage with.
  • Using C# and Unity3D to create a Random Level Generation and Object Pooling System that reduced the amount of resources needed to run the game and thus increasing mobile performance.
  • Creating a 32-page Game Design Document that detailed all the aspects of gameplay and commerciality within The Floor is Lava. This included player retention, monetisation strategy, commercial placement and business model etc.
  • Implementing and configuring Google Play In-App Purchases and Unity Advertisements which allows players to purchase items in-game and gain in-game currency by watching ads.
  • Creating a Sprite Hot-Swapping System which facilitated players being able to switch Characters throughout the game. (However, this was later replaced by a more efficient plugin)
